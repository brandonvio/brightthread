Chat Completions

OpenAI-compatible chat completions endpoint.

Args: request: Chat completion request with messages auth: Authenticated user from bearer token agent: Injected CX order support agent conversation_service: Injected conversation service

Returns: Chat completion response in OpenAI format

Authorizations:
HTTPBearer
Request Body schema: application/json
required
model
required
string (Model)

Model identifier

required
Array of objects (Messages)

Conversation messages

temperature
number (Temperature) [ 0 .. 2 ]
Default: 1

Sampling temperature

max_tokens
integer (Max Tokens) > 0
Default: 1024

Maximum tokens to generate

stream
boolean (Stream)
Default: false

Whether to stream responses

Session Id (string) or Session Id (null) (Session Id)

Optional session ID for conversation continuation

order_id
required
string (Order Id)

Order ID for order support context
